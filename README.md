# Juan Pablo CV Web

CV personal estilo elite construido con Next.js App Router, Tailwind CSS y Framer Motion, listo para ejecutar en GitHub Codespaces o desplegar en Vercel.

## Requisitos

- Node.js 20.9 o superior
- npm

## Ejecutar en local o Codespaces

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Desplegar en Vercel

1. Sube esta carpeta a un repositorio en GitHub.
2. Importa el repositorio en Vercel.
3. Vercel detectará automáticamente que es un proyecto Next.js.
4. Ejecuta el deploy.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- Lucide React
